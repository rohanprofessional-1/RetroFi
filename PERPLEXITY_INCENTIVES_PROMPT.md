# Perplexity Prompt: State Financial Incentives for Sustainable Households

## HOW TO USE THIS FILE

This file contains a **master research prompt** and **49 per-state prompts** (all US states except Georgia, which is already covered).

**Recommended workflow:**
1. Open Perplexity at [perplexity.ai](https://perplexity.ai) and enable **Pro Search** for deeper sourcing.
2. Start with the **Master Discovery Prompt** to get an overview.
3. Then run each **Per-State Prompt** one at a time. For each state, Perplexity will return direct links — download the PDFs and save the HTML pages to the `sources/` folder.
4. Name each file clearly: `[STATE]-[Program Name].[pdf|html]`

**Target document types (matching existing sources/):**
- Downloadable PDFs from state energy offices, public utility commissions, or state housing agencies
- Official government HTML pages describing rebate/credit programs
- IRS or DOE program-specific guidance documents
- Utility rebate application forms or program factsheets

**Priority topics (in order):**
1. Solar panel installation credits and rebates
2. Home electrification rebates (IRA HOMES / HEAR programs administered by the state)
3. Heat pump (HVAC + water heater) incentives
4. Energy efficiency upgrades (insulation, air sealing, windows, doors)
5. EV charger credits
6. Low-income weatherization assistance

## MASTER DISCOVERY PROMPT

Copy and paste this into Perplexity first to identify which states have the richest incentive landscapes:

```
I am building a database of state-level financial incentives for sustainable and energy-efficient households across all 50 US states. I need to find official government documents — specifically downloadable PDF factsheets, program guides, and official state/government web pages — that describe these incentives.

Focus on incentives in these categories, in priority order:
1. Solar panel installation tax credits or rebates (state-level, not federal IRS)
2. Home Electrification and Appliance Rebates (HEAR) — IRA-funded programs administered by individual states
3. Home Owner Managing Energy Savings (HOMES) rebates — IRA-funded, state-administered
4. Heat pump (space heating/cooling and water heater) incentives
5. Home insulation, air sealing, windows, and doors rebates
6. EV home charger credits
7. Low-income weatherization assistance programs

For each of the following states, identify the PRIMARY state agency or program administering these incentives and provide the official program website URL and any direct PDF download links:

Alabama, Alaska, Arizona, Arkansas, California, Colorado, Connecticut, Delaware, Florida, Hawaii, Idaho, Illinois, Indiana, Iowa, Kansas, Kentucky, Louisiana, Maine, Maryland, Massachusetts, Michigan, Minnesota, Mississippi, Missouri, Montana, Nebraska, Nevada, New Hampshire, New Jersey, New Mexico, New York, North Carolina, North Dakota, Ohio, Oklahoma, Oregon, Pennsylvania, Rhode Island, South Carolina, South Dakota, Tennessee, Texas, Utah, Vermont, Virginia, Washington, West Virginia, Wisconsin, Wyoming

For each state, return:
- State name
- Program name
- Administering agency
- Official program URL
- Direct PDF download link (if available)
- Brief description of what the incentive covers and maximum dollar amount

Prioritize official .gov sources, state energy office pages, and direct PDF links over third-party aggregators.
```
